Iteration

The %IteratorPrototype% Object

%IteratorPrototype%.map ( _mapper_ )

1. If IsCallable(_mapper_) is *false*, throw a *TypeError*. 1. Let _iterated_ be GetIteratorDirect(_this_). 1. Let _iterator_ be ObjectCreate(%IteratorPrototypeMapIteratorPrototype%, « [[Mapper]], [[Iterated]] »). 1. Set iterator.[[Mapper]] to _mapper_. 1. Set iterator.[[Iterated]] to _iterated_. 1. Return _iterator_.

%IteratorPrototypeMapIteratorPrototype%

The %IteratorPrototypeMapIteratorPrototype% object:

  • has a [[Prototype]] internal slot whose intrinsic value is %IteratorPrototype%.
  • is an ordinary object.
